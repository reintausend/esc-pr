# printer_setup/ – ESC/POS-Modul (TM-T88IV)

Portables Python-Modul für USB-Druck, Bildvorbereitung und Papierabschnitt.

| Inhalt | Zweck |
|--------|--------|
| `PRINTER_SETUP.md` | Ausführliche Dokumentation (Installation, Endpoints, Fehler) |
| `settings.yaml` | Druckerbreite (512 dots), USB-Endpoints |
| `discover_usb.py` | USB-Geräte finden |

Aufgerufen von `Printer/print_server.py`. Nicht Teil des Browsers/Editors.
